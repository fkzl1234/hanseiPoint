한세 과제용 탬플릿 '한세포인트'
포렌식 프로젝트 클론코딩입니다.
----
>주어진 HTML,CSS를 활용해 포인트 적립페이지,포인트 적립 성공페이지, 로그인,회원가입 페이지에 기능을 구현하였습니다.
>Java Spring boot를 이용해 벡엔드를 개발 및 연동하였고 상품 구매 기능과 취소기능도 구현하였습니다.
